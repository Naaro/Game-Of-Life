Infinity Mirror

This is an infinity mirror I made during the COVID times!

Using an arduino (ESP8266) Node MCU, a string of 73 5v LEDS, a mirror and some 40% transparancy acrylic I was able to create an infinity effect and the results are pretty cool!

Along with this project I made a phone app which can control the Infinity Mirror in different ways, such as changing the color (Red Green Orange Yellow Purple...) and setting the mode (Wave Solid Bar Pulse Off...)

Electric Longboard

I have made 2 Longboards now and both are extremely fun to ride around on.

The first board I made used hub motors which are built into the wheels and work using magnets. This was an easier way to start out and is probably how I would recommend starting to anyone new and interested. This first board I made very loose and am able to make pretty sharp turns with. It reminds me a lot of snowboarding, and because I used hub motors it's pretty quiet too.

For my second board I got a little more hands on. This board is belt driven and quite a bit more involed. The battery for this board was hand built, the trucks and motors had to be put together in a way that allowed for clearance between the ground and the motors (which was a struggle in my case). This board is much stiffer than my first one and louder, but it has way more torque and i've unofficially raced a few cars and won so it's working out fine.

This was an adult legos type project so no coding was required, however the Vesc (Variable ESC) that I used for my second board allows for reading and writing values to the ESC and I have plans to bevel out and attach LEDs to the bottom of the board for an underglow effect based on your current speed. I'll release code for that when I get around to it.

Smart Mirror

So this is a smart mirror (I know, I've done a lot with mirrors). I originally made this because I had an old monitor that I didnt know what to do with and this is what it became!

Including the monitor I also used 1 Raspberry PI 3, an arduino, and a few slats of wood from under my sisters bed (she never noticed). As you can see there's some bootup time which is the PI doing its initial load, during which the arduino is running a "loading" loop for the LEDs. Once the PI completely boots up, it kicks off some Python code and sends a signal to the arduino telling it we're now booted and the arduino changes to a blue glow. The PI connects to Wifi and will pull down current date and weather information along with recent news articles and can really be configured to display any mannor of useful information.

This is likely my favorite way to make use of old monitors ;)
